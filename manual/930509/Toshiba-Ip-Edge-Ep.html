    Chapter 1 – Introduction INTRODUCTION The IPedge™ System is an advanced unified communications systems. It consists of an Integrated IPedge Solutions server running multiple applications. A single system could be made up of one or more IPedge Servers. Multi-node IPedge Net Systems consist of multiple IPedge Solutions servers connected via IPedge Net to appear as a single large system.
