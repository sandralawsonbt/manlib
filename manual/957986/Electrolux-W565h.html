  • Page 72: Main Washer

    Troubleshooting MAIN WASHER 11:1 NO WATER This error is shown if the programmed water level is not reached within a certain time, typically 10 minutes. Max. filling time is defined in Config. 1 parameter MAX FILL TIME. This error message can be turned off in Configuration - Error code. Possible causes: Long filling times can be caused by a leaking drain valve, blocked filler valve, defective filler valve, defective valve control board, clogged level sensor hose, leaking level system, etc.
